Zadatak 28: 
Kreirati Web sever koji omogućava klijentu pretragu slika korišćenjem Rijks Museum Data 
Services API-a. Pretraga se može vršiti pomoću filtera koji se zadaju u okviru query-a. Spisak dela koja zadovoljavaju uslov se vraćaju kao odgovor klijentu. Svi zahtevi se šalju preko browsera korišćenjem GET metode. Ukoliko navedene informacije ne postoje, prikazati grešku korisniku.  
Način funkcionisanja Rijks Museum Data Services API-a je moguće proučiti na sledećem linku: 
https://data.rijksmuseum.nl/docs/search 
Primer poziva serveru: https://data.rijksmuseum.nl/search/collection?type=painting
